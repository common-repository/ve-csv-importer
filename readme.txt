Import Pages/Posts with post category from CSV files into WordPress.

== Description ==

"VE CSV Importer" plugin help to import pages/posts from a CSV (Comma Separated Value) files into your WordPress site.

= Features =


* An options for import to page through CSV files.
* An options for import to posts through CSV files.
* An option for import to post with their category name.
* An option for upload to custom meta field value through CSV.
 
== Installation ==

Installing the plugin:

1. Upload the plugin files to the `/wp-content/plugins/ve-csv-importer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tools->VE CSV importer screen to configure the plugin
